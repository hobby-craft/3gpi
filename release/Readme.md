# リリースノート

## 3gpi2-stretch-20171102  
  初期ログイン情報カードに 20171102 の記載があるものには本リリースの microSD カードが付属します。  
  詳細は、[3gpi2-stretch-20171102.md](./3gpi2-stretch-20171102.md) を参照してください。
