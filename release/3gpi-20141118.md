# 3gpi-20141118

  * 3gpi-utils を v0.1-1 から v0.2-1 に更新  
  * 3gpi-network-manager を v0.1-1 から v0.1-2 に更新  
  * その他のパッケージは 20141118 時点の最新版に更新
